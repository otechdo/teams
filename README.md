# What is it ?

It's a git commit helper with format, check source code quality before commit with commit message like AngularJS.

## Remote

Call your remote `origin`.

## Installation

```bash
cargo install zuu commiter
```

## Usage

```bash
commiter
```

# Commit Message Format

## Commit Message Header

```
<type>(<scope>): <short summary>
  │       │             │
  │       │             └─  Summary in present tense. Not capitalized. No period at the end.
  │       │   
  │       └─ Commit Scope:  auth|email-template|devops|localization|api|logging|navigation|middleware|service|
  │                         model|view|controllers|subscription|cli|ui|ux|seo|deps|lang|theme|perf|search|
  │                         payment|forms|design|router|db|lib|exe|main|perf
  │
  └─ Commit Type    Star|Comet|Nebula|Pulsar|Quasar|Asteroid Belt|Solar Flare|Dwarf Planet|Supermassive Black Hole|  
                    Eclipse|Supernova|Red Giant|White Dwarf|Neutron Star|Black Hole|Wormhole|Solar Wind|
                    Lunar Eclipse|Cosmic Microwave Background|Gravitational Wave|Event Horizon|Big Bang|Launch|Probe|Slingshot Maneuver|Wormhole|
                    Lightspeed|Mission Control|Spacewalk|Moon Landing|First Contact|Terraform|Interstellar Communication|UFO|Houston, We Have a Problem|E.T. Phone Home|
                    Warp Speed|Space Junk|Houston, We Have a Solution|Alien Technology|May the Force Be With You|Celestial Dance|
                    Cosmic Symphony|Stardust Whispers|Nebula's Embrace|Stellar Nursery|Quantum Leap|Cosmic Alignment|Warp Drive|Hyperspace Jump|First Contact|
                    Stargate|Prime Directive|Holo deck Simulation|Replicator|Cosmic Dawn|Galactic Harmony|Solar Wind|
                    Lunar Eclipse|Stellar Evolution|Nebula's Breath|Red shift|Blue shift|Parallax|Accretion Disk|Singularity| 
                    Quantum Entanglement|Dark Matter|Gravitational Lensing|Time Dilation
```

**Celestial Objects and Phenomena:**

* **Star:** Introduces a significant new feature.
* **Comet:** Resolves a fleeting bug or issue.
* **Nebula:** Creates a new project or feature from scratch.
* **Pulsar:** Enhances performance.
* **Quasar:** Improves documentation or code visibility.
* **Asteroid Belt:** Refines code formatting and appearance.
* **Solar Flare:** Adds or updates tests.
* **Dwarf Planet:** Performs routine maintenance or updates.
* **Black Hole:** Makes substantial changes to the build process.
* **Eclipse:** Temporarily disables or hides a feature.
* **Supernova:** Introduces a major overhaul or complete rewrite.
* **Red Giant:** Expands or enhances an existing feature.
* **White Dwarf:** Condenses or simplifies code.
* **Neutron Star:** Optimizes for extreme performance.
* **Black Hole:** Removes dead or unused code.
* **Wormhole:** Merges branches or connects disparate parts of the codebase.
* **Solar Wind:** Refactors code for smoother flow.
* **Lunar Eclipse:** Temporarily disables a feature for maintenance.
* **Cosmic Microwave Background:** Adds logging or monitoring.
* **Gravitational Wave:** Fixes a subtle but impactful bug.
* **Event Horizon:** Marks a point of no return or major decision.

**Space Exploration:**

* **Big Bang:** The very first commit of a project.
* **Launch:** Deploys code to production.
* **Probe:** Implements a new experimental feature.
* **Slingshot Maneuver:** Reverts a previous commit.
* **Wormhole:** Merges branches or connects disparate parts of the codebase.
* **Lightspeed:** Significantly improves performance.
* **Mission Control:** Updates project management or tracking tools.
* **Spacewalk:** Fixes a critical issue in production.
* **Moon Landing:** Completes a major milestone or release.
* **First Contact:** Integrates with a new external system or API.
* **Terraform:** Makes significant changes to the project's infrastructure.
* **Interstellar Communication:** Improves documentation or communication.

**Humorous Cosmic Terms:**

* **UFO:** Adds a mysterious or unexplained change.
* **Houston, We Have a Problem:** Signals a critical issue or bug.
* **E.T. Phone Home:** Adds or modifies communication with external services.
* **Warp Speed:** Describes a significant performance improvement.
* **Space Junk:** Removes unused code, files, or dependencies.
* **Houston, We Have a Solution:** Fixes a previously reported problem.
* **Alien Technology:** Integrates a new and innovative library or tool.
* **May the Force (Push) Be With You:** Deploys code to production.

**Poetic and Metaphorical:**

* **Celestial Dance:** Harmonious interaction between components.
* **Cosmic Symphony:** Improved user experience.
* **Stardust Whispers:** Subtle but importanta corrections.
* **Nebula's Embrace:** Harmonious integration of a new component.
* **Stellar Nursery:** Creation of a promising new project.
* **Event Horizon:** Point of no return in a change.
* **Quantum Leap:** Significant performance improvement.
* **Cosmic Alignment:** Resolution of complex conflicts.
* **Warp Drive:** Acceleration of development.
* **Hyperspace Jump:** Transition to a new project phase or technology.
* **First Contact:** Communication with an external system.
* **Terra forming:** Major infrastructure change.
* **Stargate:** Creation of a portal to a new feature or service.
* **Prime Directive:** Adherence to development best practices.
* **Holo deck Simulation:** Creation of a virtual test environment.
* **Replicator:** Automation of repetitive tasks.
* **Cosmic Dawn:** First commit of a new project.
* **Galactic Harmony:** Successful merging of code branches.
* **Solar Wind:** Performance and efficiency improvements.
* **Lunar Eclipse:** Temporary disabling of a feature.
* **Stellar Evolution:** Major code refactoring.
* **Nebula's Breath:** Adding comments and documentation.

**Scientific Concepts:**

* **Red shift:** Reverting to an older version.
* **Blue shift:** Upgrading to a newer version.
* **Parallax:** Fixing a bug caused by misinterpretation.
* **Accretion Disk:** Gradual addition of features.
* **Singularity:** Extreme code simplification.
* **Quantum Entanglement:** Creating tight links between components.
* **Dark Matter:** Adding hidden or experimental features.
* **Gravitational Lensing:** Improving visibility of a problem.
* **Time Dilation:** Optimizing code execution time.
